# HELIOS — Global Insight Globe 🌍

An AI-powered interactive orbital intelligence platform built for the Build with MeDo Hackathon by DEVPOST.

🔗 Live Demo: https://app-bc6lra5w5on5.appmedo.com

---

## 🚀 About The Project

HELIOS transforms global economic and geopolitical data into an immersive 3D intelligence experience.

Users can interact with a cinematic globe interface and explore:
- 📈 Economic insights
- 💱 Currency systems
- 📊 Stock market indicators
- 🎓 Education intelligence
- 💻 Technology ecosystems
- ✈️ Tourism analytics
- 🚀 Startup ecosystem insights
- 🧠 AI-generated tactical country briefs

The project evolved iteratively through multiple versions using MeDo’s AI-assisted development workflow.

---

## ✨ Features

- Interactive 3D rotatable globe
- Dynamic country intelligence panels
- AI tactical brief generation
- Heatmap visualization system
- Holographic HUD-inspired UI
- Animated orbital rings & atmospheric effects
- Smooth cinematic transitions
- Currency conversion system
- Multi-dimensional country analysis

---

## 🛰️ Tech Stack

TypeScript - React 18 - Vite - Tailwind CSS - react-simple-maps - Framer Motion - lucide-react - shadcn/ui - Radix UI - SVG Filters - React Hooks - Supabase - Biome - pnpm - CSS Keyframes - SVG SMIL
